# Portfolio — Pablo del Barrio

## Introduction

Personal portfolio built with Next.js 16, featuring a terminal-inspired UI, responsive design, and full internationalization (ES/EN).

## Features

- **Minimal terminal design** — retro aesthetic with green-on-dark palette
- **Responsive** — adapts seamlessly from desktop to mobile
- **Internationalization (i18n)** — Spanish and English via next-intl
- **Keyboard navigation** — navigate sections with arrow keys
- **Projects showcase** — interactive list/detail view with video modals

## Technologies Used

- **Next.js 16** (App Router, RSC)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **next-intl**
